Models
Using your own models
You can add your own GGUF models manually.
Format: GGUF only (.gguf files)
Location:
Naming: Any filename works. The app auto-detects Phi-3 and Llama 3 chat formats from the filename. Other models use Llama 3 format by default.
Tip: Q4_K_M quantizations offer the best balance of speed and quality.
